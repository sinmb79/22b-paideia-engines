# Example Data Index

[한국어](example_data_index.ko.md)

This repository includes only safe sample data and source metadata.

| Path | Purpose | Public Release Status |
| --- | --- | --- |
| `data/catalog/seed_sources.json` | Metadata for official or licensable education sources | Allowed |
| `data/curriculum/sample_standards.json` | Small sample curriculum standards | Allowed |
| `examples/configured_suite.json` | Sample config for the Phase 5 runner | Allowed |
| `examples/acquired_sources_manifest.jsonl` | Public-safe acquired-source manifest diagnostics sample | Allowed |
| `examples/public_item_sample.jsonl` | Tiny repository-owned item sample used by acquisition validation examples | Allowed |
| `examples/source_fixture_pack.json` | Public-safe parser diagnostics fixture manifest | Allowed |
| `examples/stress_packs/core_subject_stress_pack.json` | Public-safe subject stress scenario pack | Allowed |
| `examples/source_samples/ncic_curriculum_sample.csv` | Tiny NCIC-style curriculum parser sample | Allowed |
| `examples/source_samples/aihub_math_sample.json` | Tiny AI-Hub-like parser sample | Allowed |
| `examples/source_samples/ebsi_exam_metadata_sample.csv` | Metadata-only public exam parser sample | Allowed |

## Not Included

- Full textbooks
- Purchased publisher materials
- AI-Hub downloaded corpora
- Personal documents
- Private voice or image assets
