# Example Data Index

This repository includes only safe sample data and source metadata.

| Path | Purpose | Public Release Status |
| --- | --- | --- |
| `data/catalog/seed_sources.json` | Metadata for official or licensable education sources | Allowed |
| `data/curriculum/sample_standards.json` | Small sample curriculum standards | Allowed |
| `examples/configured_suite.json` | Sample config for the Phase 5 runner | Allowed |
| `examples/public_item_sample.jsonl` | Tiny repository-owned item sample used by acquisition validation examples | Allowed |

## Not Included

- Full textbooks
- Purchased publisher materials
- AI-Hub downloaded corpora
- Personal documents
- Private voice or image assets
