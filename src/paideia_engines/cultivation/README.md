# Cultivation Engine

[한국어](README.ko.md)

The Cultivation Engine builds training blueprints and curriculum-linked learning roadmaps.

## Owns

- Learner blueprints
- Learning roadmaps
- Source policy summaries
- Assessment gate placement

## Public API

- `CultivationEngine(...)`
- `create_blueprint(...)`
- `build_learning_roadmap(...)`

## Safety Boundary

This engine designs growth paths. It does not grade results or admit memory.
